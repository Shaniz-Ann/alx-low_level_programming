# 0x00. C - Hello, World

## 📋 Project Overview

This is my **personal revision** of the ALX Low Level Programming curriculum. This project introduces the fundamentals of C programming, compilation process, and basic output functions.

---

## 🎯 Learning Objectives

By completing this project, I have demonstrated understanding of:

### General Concepts
- Why C programming is awesome
- History: Dennis Ritchie, Brian Kernighan, Linus Torvalds
- The C compilation process
- Entry points and the `main` function
- Printing text using `printf`, `puts`, and `putchar`
- Getting type sizes with `sizeof`
- Compiling with `gcc` and its default behavior
- Betty coding style
- Finding correct headers for standard library functions
- Return values of the main function
